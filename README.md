# Flux

App de rede social desenvolvido para o projeto prático de Dispositivos Móveis 2 (ARQDMO2) do IFSP.

## Sobre o Projeto
O Flux é um app simples onde você pode postar fotos, escrever uma descrição e marcar a localização de onde está postando. O projeto foi feito em Kotlin e usa o Firebase para guardar os dados e fazer o login.

## Funcionalidades
- **Login e Cadastro:** Dá para criar conta e logar usando e-mail e senha (Firebase Auth). O login é salvo automaticamente.
- **Postagens:** Você pode criar um post com uma imagem da galeria, texto e a sua cidade atual.
- **Feed:** O app puxa os posts salvos do banco e exibe de 5 em 5.
- **Busca:** Tem uma barra para pesquisar os posts filtrando pelo nome de uma cidade específica.
- **Perfil:** Na tela de perfil dá para alterar o nome, trocar a foto e atualizar a senha.
- **Localização:** Quando você vai criar um post, o app puxa sua localização via GPS e descobre o nome da cidade.
